Soul Foods Sales Analytics Project

Quantium Data Engineering Assessment

Project Overview

This project was developed as part of a data engineering and visualization assessment for Soul Foods, a fictional food company. The goal was to process transaction data for their Pink Morsels product line, build an interactive dashboard to visualize sales trends, and implement automated testing for continuous integration.

Business Question: "Were sales higher before or after the Pink Morsel price increase on the 15th of January, 2021?"

Data Processing & ETL Pipeline

  • ETL pipeline to clean and merge 3 CSV datasets (daily_sales_data_0.csv, daily_sales_data_1.csv, daily_sales_data_2.csv)
  • Filtered for Pink Morsels transactions only
  • Cleaned price data (removed $ symbols and redundant values)
  • Calculated sales by multiplying quantity and price
  • Extracted required columns: sales, date, region
  • Consolidated into single dataset: final_sales.csv

Interactive Dashboard

  • Built with Dash and Plotly for interactive web visualization
  • Line chart showing sales trends over time with date-sorted data
  • Region filter (North, South, East, West, All) with radio buttons
  • Real-time updates based on user selection
  • Custom CSS styling for professional interface
  • Header: "Pink Morsels Sales Visualizer"

Automated Testing & CI/CD

  • Test automation with pytest and Dash testing utilities
  • Test Cases:
    • Header Presence: Verifies dashboard title is displayed
    • Visualization Presence: Ensures line chart component is rendered
    • Region Picker Presence: Confirms all five radio button options are available
  • All tests pass successfully
  • CI-ready scripts (run_tests.ps1 / run_tests.sh) for automated test execution
  • Can be integrated into CI/CD pipelines (GitHub Actions, Jenkins)

Key Insights & Findings

  • Sales Impact: Noticeable drop in sales immediately after January 15, 2021 price increase
  • Regional Analysis: Trend consistent across all regions
  • Most Affected: Southern region experienced the most pronounced decline
  • Conclusion: Price increase had negative impact on sales volume
  • Visual Evidence: Clear demonstration of price elasticity impact
